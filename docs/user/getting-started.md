# Getting Started — Nova Rewards

Welcome to Nova Rewards. This guide helps you get set up quickly so you can start earning and redeeming rewards.

## Overview

Nova Rewards lets users earn on-chain rewards and redeem them through the web dashboard.

## Prerequisites

- A modern browser (Chrome, Edge, Firefox, Brave, Safari)
- A supported wallet (see the wallet connection guide)
- Network access to the app (staging or production URL provided by the platform)

## Quick Start

1. Open the Nova Rewards web app URL provided by your organization.
2. Connect your wallet (follow the wallet connection guide).
3. If you don't have an account in the UI, follow the on-screen flow to register or accept the terms.
4. Visit the Dashboard to view your earned rewards, balances, and available redemptions.

## Common Actions

- View active campaigns: open the `Campaigns` or `Earn` page in the UI.
- Check balances: `Wallet` or `Balances` panel shows on-chain balances and pending rewards.
- Redeem rewards: see the Rewards Redemption Walkthrough.

## Where to get help

- For technical issues or missing funds, contact support or open an issue in the project repository.

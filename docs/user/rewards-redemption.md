# Rewards Redemption Walkthrough

This walkthrough shows the typical steps to redeem rewards from the Nova Rewards web dashboard.

## Before you start

- Make sure your wallet is connected and on the correct network.
- Ensure you have enough balance to pay any on-chain transaction fees.

## Step-by-step redemption

1. Open the Nova Rewards dashboard and navigate to the `Redeem` or `Rewards` page.
2. Find the reward item you want to redeem. Items may show estimated value, availability, and any redemption limits.
3. Click the `Redeem` button for the chosen reward.
4. Review the redemption details in the confirmation dialog: amount, destination (on-chain or off-chain), and any fees.
5. Approve the transaction in your connected wallet when prompted. Confirm gas/fee estimates before approving.
6. Wait for the transaction to be included in a block. The UI will show a pending status and then confirmed when complete.
7. After confirmation, check the `History` or `Transactions` page to verify the redemption.

## Off-chain redemptions (gift cards, partner services)

- Some redemptions may trigger an off-chain fulfillment flow. You will receive an email or UI notification with next steps and redemption codes.

## Troubleshooting

- Transaction failed: check wallet error message, ensure sufficient funds for fees, retry if needed.
- Redemption not received: verify transaction confirmation on the block explorer and contact support with the transaction hash.

## Tips and best practices

- Confirm the destination address before approving any transaction.
- Keep a copy of transaction hashes for support requests.
